This is the file containing the modification database for Proteax Desktop: http://www.biochemfusion.com/products/proteax_desktop/

Follow installation instructions obtained with Proteax Desktop

To install/update modification database on Linux

cd $PROTEAX_DESKTOP_5_HOME
wine bin/ProteaxAdmin.exe

select the current modification database (found in $PROTEAX_DESKTOP_5_HOME/db/proteax_desktop_mod_db.database)

select "File" = > "Import from file..."
Select the proteax_desktop_mod_db_export.sdf file
click "Import"
click "Accept import"
